# Changelog - 智能问答助手

## [2026-03-27] - 引用跳转与高亮功能增强

### 新增功能

#### 1. 📌 点击引用自动跳转 PDF 页码
- **问题修复**: 之前点击引用时，右侧 PDF 预览不会跳转到对应页面
- **实现方式**:
  - 从 chunk 文本中智能提取页码信息（格式：`# 第 X 页`）
  - 点击引用按钮时，自动设置 `selected_page` 为目标页码
  - PDF 预览面板自动刷新并跳转到指定页面

#### 2. 🎯 检索内容高亮显示
- **功能描述**: 在 PDF 预览中以黄色高亮显示检索到的文本片段
- **技术实现**:
  - `find_text_in_page()`: 在 PDF 页面中搜索匹配文本，返回矩形区域
  - `render_pdf_preview()`: 渲染 PDF 时添加高亮注释
  - 高亮颜色：黄色半透明 (`stroke=(1, 1, 0, 0.5)`)
- **搜索策略**:
  1. 首先尝试完整文本匹配
  2. 其次尝试清理后的文本（移除多余空白）
  3. 最后逐句搜索（按。！？分割）

#### 3. 📋 高亮状态管理
- **新增会话状态**:
  - `highlight_text`: 要高亮的文本内容
  - `active_citation`: 当前激活的引用 ID
- **UI 反馈**:
  - 显示"🔍 已启用高亮显示"状态提示
  - 成功匹配时显示"✅ 在当前页面找到 X 处匹配"
  - 未匹配时显示"⚠️ 当前页面未找到匹配文本"
  - 提供"清除高亮"按钮

### 使用方式

1. **启动应用**
   ```bash
   streamlit run app.py
   ```

2. **查看引用并跳转**
   - 在聊天框提问
   - AI 回答后，展开"📚 查看引用来源"
   - 点击任意引用按钮（如 `📄 [1] document.pdf`）
   - 右侧 PDF 预览自动跳转到对应页面

3. **查看高亮内容**
   - 点击引用后，检索到的文本片段会在 PDF 中以黄色高亮显示
   - 如果当前页面没有找到匹配文本，会显示提示并可切换页码查找
   - 点击"清除高亮"可取消高亮显示

### 注意事项

1. **页码提取依赖**: chunk 文本需要包含 `# 第 X 页` 格式的标记
   - 使用 `ingest_lite.py` 索引的 PDF 自动包含此标记

2. **高亮匹配限制**:
   - 如果 PDF 文本经过特殊编码或扫描版，可能无法精确匹配
   - 建议优先使用文字版 PDF

3. **性能考虑**:
   - 高亮功能需要在渲染时搜索文本，大 PDF 可能略有延迟
   - 可点击"清除高亮"恢复普通预览模式
